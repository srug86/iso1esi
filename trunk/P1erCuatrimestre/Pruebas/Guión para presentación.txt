GUIÓN:
------
Crear modelos de evaluación
---------------------------
1. Entramos al sistema como secretario (5699591).
2. Añadir modelo de evaluación.
3. Elegir un paquete de proyectos.
4. Añadimos secciones.
5. Guardamos modelo de evaluación.
6. Modificar modelos de evaluación.
7. Eliminar el modelo creado.

Visionado general de los adjuntos
---------------------------------
1. Adjunto Subárea 4 (5701903) -> Dos expertos están realizando evaluaciones de un mismo proyecto.
2. Adjunto Subárea 2 (3421588) -> Un experto está realizando evaluaciones de dos proyectos.
3. Adjunto Subárea 1 (5692919) -> Dos expertos están realizando cada uno la evaluación de un proyecto distinto.

Realizar evaluación por experto
-------------------------------
1. Entramos al sistema como experto (5702008)
2. Seleccionar proyecto asignado.
3. Realizar evaluación. (Ver evaluación)
4. Guardamos evaluación.
5. Modificamos evaluación pulsado realizar evaluación.
6. Finalizamos evaluación (ver que el estado de la evaluación cambia a "Terminado").
7. Entramos en el coordinador (5692939)
8. Vemos el estado de los proyectos que tiene asociado (el coordinador).

Gestión del adjunto
-------------------
1. Entramos al sistema como adjunto (5701903)
2. Se puede observar todos los proyectos asociados a este adjunto con el 
determinado estado de la evaluación del proyecto.
3. Para ver los expertos asociados, se pulsa '+'.
4. Aparecerán los expertos asignados a ese proyecto junto con el estado de la evaluación de cada uno de ellos.
5. Vemos que mientras estado del proyecto de 5702008 está "evaluado por el experto", el estado del proyecto de 70582355 está "en proceso".
6. Se selecciona un proyecto y se realiza el informe final.
7. Guardamos informe final.
8. Entramos en experto 5702008 para demostrar que no se puede modificar la evaluación.
9. En adjunto, finalizamos informe final.

Validar por el coordinador
--------------------------
1. Entramos al sistema como coordinador (5692939).
2. Seleccionar un proyecto para ver/modificar el informe.
3. Validar informe.
4. Entramos en adjunto (5701903).
