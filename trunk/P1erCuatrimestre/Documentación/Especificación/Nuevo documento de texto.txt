1.1 Listado de usuarios --> se encarga de dados unos parámetros de consulta, ya pueden ser introducidos implícitamente por el usuario o por el propio sistema, ya que dependiendo de quién realice la consulta el sistema le dará acceso a unos datos u otros. Al realizar esta consulta el proceso consulta los almacenes de datos SUBÁREAS y USUARIOS, ya que devuelve los datos organizados según la subárea en que trabajen dichos usuarios del sistema. La relación jerárquica del sistema decide a qué usuarios se tiene acceso, es decir, si es el Secretario el que pide el listado este podrá ver todos los datos, si es el Coordinador podrá ver todos los adjuntos de su Área y todos los expertos y si es el adjunto sólo podrá acceder a los usuarios, mientras que los expertos no tienen esta opción.
1.2 Añadir Usuarios --> este proceso es exclusivo para el secretario, ya que es el único con privilegios para introducir nuevos usuarios en el sistema.  El sistema recibe una nueva instancia de usuario
